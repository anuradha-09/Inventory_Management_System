# 📦✨ Inventory Management System

A complete **Inventory Management System** built using **ASP.NET MVC**, **MySQL**, and **React**.  
The application supports full CRUD operations through standard ASP.NET MVC action methods (GET, POST, PUT, DELETE) and provides a clean frontend interface.

---

## ⭐ Features

- 📋 Add, update, delete, and view products  
- 🧾 Manage suppliers  
- 📦 Track stock levels  
- 🟦 ASP.NET MVC controllers with GET/POST/PUT/DELETE  
- 🐬 MySQL database  
- ⚛️ React-based frontend
